O que é CSS3?

CSS3 é a terceira versão da linguagem de estilo em cascata, utilizada para descrever a apresentação de documentos HTML e XML. Ele permite que os desenvolvedores criem layouts complexos e estilizações atraentes para suas páginas web.

Propriedades de Background

O CSS3 introduziu várias propriedades para manipular backgrounds de forma mais eficaz:

  • background-color: Define a cor de fundo de um elemento.
  • background-image: Define uma imagem como plano de fundo.
  • background-position: Posiciona a imagem de fundo (ex: center, top, bottom).
  • background-repeat: Controla se a imagem de fundo deve se repetir.
  • background-size: Especifica o tamanho da imagem de fundo (ex: cover, contain).
  • background-attachment: Define como a imagem de fundo se comporta em relação ao rolar da página.
  • background-blend-mode: Mescla a cor de fundo com a imagem, criando efeitos interessantes.
